# Veltora

An online storefront for daily essentials, trip gear, accessories, and gadgets. Built with React and Tailwind CSS.

## What it includes

- Homepage with hero section
- Product grid with category filtering (Daily Practicals, Trip Essentials, Accessories, Gadgets)
- Search bar
- Shopping cart drawer with quantity controls
- Responsive mobile layout

## Tech Stack

- React
- Tailwind CSS
- Lucide React (icons)
- Google Fonts (Space Grotesk, Source Serif 4, JetBrains Mono)

## Getting Started

```bash
npm install
npm run dev
```

## Planned Integrations

- Supabase (database + auth)
- Stripe (payments)
- Vercel (hosting)
